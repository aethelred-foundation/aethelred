# Support

## Where To Start

- Documentation: [docs.aethelred.io](https://docs.aethelred.io)
- Canonical chain repository: [aethelred-foundation/aethelred](https://github.com/aethelred-foundation/aethelred)
- Governance proposals: [aethelred-foundation/AIPs](https://github.com/aethelred-foundation/AIPs)
- Release provenance: [docs/security/release-provenance.md](docs/security/release-provenance.md)

## Questions And Usage Help

- Use GitHub Issues for reproducible bugs and actionable feature requests.
- Use GitHub Discussions where enabled for design discussion and Q&A.
- Use [Discord](https://discord.gg/aethelred) for community support and maintainer routing.

## Security

Do not open public issues for vulnerabilities.

- Security reporting: [security@aethelred.io](mailto:security@aethelred.io)
- Security policy: [SECURITY.md](SECURITY.md)

## Commercial / Partner Inquiries

- Foundation contact: [foundation@aethelred.io](mailto:foundation@aethelred.io)

## Response Expectations

- Security reports: acknowledgement within 24 hours
- Bug triage: initial maintainer response target within 3 business days
- General support: best effort via GitHub or Discord
