### Operating Model

- `AethelredFoundation/aethelred` is the canonical public source of truth.
- Standalone repos exist for packaging, discoverability, and focused contributor workflows.
- Public audit and release claims are backed by linked evidence, not marketing-only badges.
- `.github` provides org-wide defaults for conduct, support, and issue intake.
