# Aethelred Dashboard

Web-based blockchain explorer and dashboard for Aethelred AI Blockchain.

## Features

- Real-time Network Stats
- Job Explorer
- Seal Viewer & Verification
- Model Registry Browser
- Validator Dashboard
- Network Activity Charts

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

```env
NEXT_PUBLIC_RPC_URL=https://rpc.mainnet.aethelred.org
NEXT_PUBLIC_API_URL=https://api.mainnet.aethelred.org
```
