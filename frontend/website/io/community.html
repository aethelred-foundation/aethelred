Contributing

Contributing to Aethelred

We welcome contributions from developers across all experience levels.

How to contribute

  • Fork the repository and create a feature branch.
  • Use commit format type(scope): description.
  • DCO sign-off is required on all commits.
  • Each PR requires at least one approving review.
Go: gofumpt Rust: cargo fmt Solidity: forge fmt

Development setup

terminal 
git clone https://github.com/aethelred-foundation/aethelred.git
cd aethelred
make build
make test
make install
