Get Started

Quick Start Guide

Get your full node running in under 30 minutes.

Full Node Setup

1

Install Aethelred Client

Download and install the official Aethelred node software.

curl -sSL https://install.aethelred.io | bash
2

Initialize the Node

Create your node identity and configure the data directory.

aethelred init --datadir ~/.aethelred --network mainnet
3

Start Syncing

Connect to the network and begin synchronizing the blockchain.

aethelred start --syncmode snap --http --http.api eth,net,aethel
4

Monitor Your Node

Check sync status and node health.

aethelred attach --exec "aethel.syncing"
