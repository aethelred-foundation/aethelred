## Document Control Rules

1. Every document must carry: Owner, Version, Reviewer, Status, Last Updated
2. Status progression: `Scaffold` → `Draft` → `Under Review` → `Approved`
3. No document may skip a status level
4. All status changes must be logged with date, actor, and rationale
5. "Approved" status requires sign-off from both legal counsel and CSP
