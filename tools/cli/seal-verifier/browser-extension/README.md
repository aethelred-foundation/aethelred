# Aethelred Seal Verifier Browser Extension (MV3)

Offline verification popup for pasted Aethelred Digital Seal JSON payloads.

## Install (Developer Mode)

1. Open `chrome://extensions` (or your Chromium browser equivalent)
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select:
   `$AETHELRED_REPO_ROOT/cli/seal-verifier/browser-extension`

## Capabilities

- Paste seal JSON export
- Offline structural validation
- Expiry checks
- Validator-attestation count checks
- Deterministic SHA-256 fingerprint generation
